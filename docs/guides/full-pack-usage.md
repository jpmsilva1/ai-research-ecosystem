# Guia de Uso: Full Pack (Engenharia + Acadêmico)

O **Full Pack** transforma a sua interface de LLM num verdadeiro "Departamento de TI" em uma caixa, te dando acesso a ferramentas de DevOps, Engenharia de Dados, MLOps e testes automatizados.

Aqui estão alguns workflows épicos que você pode realizar com as 130+ skills habilitadas:

## 1. MLOps e Tracking de Modelos
Você rodou um modelo no seu projeto Python e quer documentar tudo profissionalmente.
- **Como usar:** Diga *"Atue como `mlops-engineer`. Revise este diretório e crie um pipeline local de MLflow para rastrear os experimentos que eu estou rodando, isolando hiperparâmetros de métricas."*

## 2. Arquitetura Orientada a Testes (TDD)
Você precisa criar uma nova funcionalidade no projeto de forma extremamente robusta.
- **Como usar:** Diga *"Ative a skill `tdd-orchestrator`. Precisamos criar um parser de JSON novo. Escreva o teste que vai falhar primeiro, e aguarde minha confirmação para implementar o código."*

## 3. Auditoria de Segurança Rápida
- **Como usar:** Peça *"Rode a skill `aegisops-ai` sobre esse meu arquivo Dockerfile e docker-compose.yml para apontar brechas de segurança, uso inadequado de permissão root ou configuração de rede falha."*

## 4. Geração Automática de Apresentações (Seminários e Aulas)
Sua pesquisa acadêmica (do Core Pack) deu frutos e você precisa criar slides em PowerPoint baseados nos seus relatórios do Obsidian.
- **Como usar:** *"Use a skill `2slides-ppt-generator` para transformar as anotações do meu arquivo `log-experimento.md` em uma estrutura de slides impactantes para apresentação."*

Explore a pasta `~/.gemini/config/skills/` (ou a pasta do Claude) para descobrir todas as outras ferramentas disponíveis!
